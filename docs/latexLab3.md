## Блочная формула

$$
\sqrt{ \frac{ \alpha^2 + \beta_i }{ \gamma^{n+1} } }
$$

## Использованные конструкции

1. **Дробь**:  $\frac{a}{b}$  
2. **Корень**: $\sqrt{x}$  
3. **Греческие буквы**:  $\alpha, \beta, \gamma$  
4. **Степень**:  $x^2$  
5. **Индекс**:  $x_i$

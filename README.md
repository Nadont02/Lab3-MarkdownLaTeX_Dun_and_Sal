## Примеры Markdown

### Форматирование текста
 
**полужирный**, *курсив*, ~~зачёркнутый~~  `моноширный` 

### Списки

**Маркированный список:**
- Элемент 1
- Элемент 2
  - Вложенный элемент
  - Ещё один

**Нумерованный список:**
1. Первый шаг
2. Второй шаг
   1. Подшаг A
   2. Подшаг B
3. Третий шаг

### Цитата

> Волк не волк, когда волк это заяц

### Блок кода

Пример отправки изменений в репозиторий:

```bash
git add .
git commit -m "Обновил README"
git push origin main
```

### Таблица

| Команда           | Назначение                     | Пример использования       |
|:------------------|:-------------------------------|:---------------------------|
| `git init`        | Инициализация репозитория      | `git init`                 |
| `git checkout -b` | Создание новой ветки           | `git checkout -b feature`  |
| `git restore`     | Отмена изменений в индексе     | `git restore --staged file`|
